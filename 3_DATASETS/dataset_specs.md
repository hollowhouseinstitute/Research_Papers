# Dataset Specifications

This document defines **minimum specifications** required for any dataset in this repository.

## Required Metadata

Every dataset must declare:
- Dataset name
- Version
- Date of creation
- Authors / curators
- Intended use
- Explicit non-uses

## Structural Requirements

- Stable directory layout
- Immutable raw data
- Versioned annotations
- Clear separation between data and interpretation

## Documentation Requirements

Each dataset must include:
- Collection or experiment design
- Annotation protocol
- Known limitations
- Ethical considerations

## Versioning

- Major versions indicate schema changes
- Minor versions indicate annotation or data expansion
- Patch versions indicate corrections only

Undocumented changes are prohibited.
