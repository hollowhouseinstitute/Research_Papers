This directory defines **standards, schemas, and protocols** governing all datasets used in Hollow House Institute research.

Datasets are treated as:
- Structured research instruments
- Ethically constrained artifacts
- Reproducible inputs to analysis

## Principles

- Datasets do not define theory
- Datasets must be interpretable without anthropomorphism
- Dataset documentation is mandatory

## Required Files per Dataset

Each dataset directory must include:
- Experiment or collection design
- Annotation protocol
- Ethical constraints
- Schema reference

Non-compliant datasets are invalid.
