# Forensic Linguistics Dataset Schema

This schema governs datasets intended for **linguistic structure analysis**, attribution, or comparative examination.

## Purpose

Forensic linguistic datasets are used to analyze:
- Structural patterns
- Stylistic markers
- Consistency and drift

They are **not** used for:
- Psychological diagnosis
- Intent inference
- Identity claims beyond structure

## Core Fields

Each entry may include:
- Text sample
- Timestamp
- Context descriptor
- Source class (synthetic / human / mixed)
- Structural features (length, syntax, repetition)

## Prohibited Fields

Datasets must NOT include:
- Mental state labels
- Emotional attributions
- Personality traits
- Speculative intent

## Usage Constraints

Results must be reported as:
- Probabilistic
- Structural
- Non-identifying

Certainty claims are prohibited.
